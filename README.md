# Система моніторингу повітря в реальному часі

Веб-орієнтований програмний комплекс для автоматизованого збору та візуалізації ключових екологічних показників атмосферного повітря в режимі реального часу.

## Функціональні можливості:
- **Автономний конвеєр даних:** автоматичний збір показників із зовнішніх API та збереження їх у хмарну базу даних.
- **Моніторинг «на зараз»:** відображення поточних значень концентрації CO2, дрібнодисперсного пилу PM2.5 та атмосферного тиску.
- **Кольорова трирівнева індикація:** миттєве візуальне оцінювання стану повітря за допомогою статусів «Чудово» (зелений), «Посередньо» (жовтий) та «Небезпека» (червоний).
- **Аналіз динаміки:** інтерактивні лінійні графіки зі спливаючими підказками (tooltips) для аналізу ретроспективних даних за обраний період (1 год, 6 год, 24 год або 7 днів).
- **Експорт звітів:** миттєве завантаження накопичених історичних даних у форматі CSV для подальшого аналізу.

## Технологічний стек:
- **Бекенд:** Node.js, Express, Mongoose (автономні сервіси та REST API).
- **База даних:** MongoDB Atlas (хмарне сховище для збереження часових рядів).
- **Фронтенд:** React, Tailwind CSS (адаптивний інтерфейс з умовною стилізацією).
- **Графіки та інструменти:** Recharts, Axios.
- **Хостинг та деплой:** Render (хостинг серверної частини), Vercel (хостинг клієнтської частини та глобальна CDN-доставка).
